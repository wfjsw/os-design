# 内存模型

## 地址空间

| 类型    | 地址范围 | 长度 |
| ------ | -------- | ---- |
| Flash | 0x08000000 - 0x0807FFFF | 512K |
| SRAM | 0x20000000 - 0x2000FFFF | 64K |

## Flash 地址分配

| 类型 | 地址范围 | 长度 |
| --- | --- | --- |
| 引导程序 | 0x08000000 - 0x080003FF | 1024B |
| 操作系统 | 0x08000400 - 0x080103FF | 64K |
| 用户程序 | 0x08014000 - 0x08053FFF | 256K |
| 保留 | 0x08054000 - 0x0007FFFF | 176K |

## 引导程序入口点

调试：0x20000000 (SRAM)  
生产：0x08000000 (Flash)
